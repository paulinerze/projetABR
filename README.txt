Programme permettant de construire et de parcourir un arbre binaire de recherche.

Limites du programme:

Le programme n'importe pas un arbre qui ne contient qu'une valeur (erreur de pointeur).
Nous vous conseillons d'essayer avec l'exemple suivant: (déjà présent dans test.txt, fichier que vous pouvez utiliser pour tester le programme)

1:6;3:2:1
9:22;9:14:11
50:75;60:55:62:72
78:80;80:80

De plus, il n'est pas possible de supprimer le dernier élément d'un ABR contenu dans une case: erreur de pointeur.
Méthode pous supprimer la dernière valeur: fusionner la case avec celle d'après (ou d'avant si c'est la dernière borne) et supprimer manuellement la valeur.
